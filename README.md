# Business Performance & Profitability Analytics

Power BI dashboard built on real Toniq retail data to track $30K+ in profit potential.

## What it does
- Tracks profitability margins (36% avg) across product categories
- Dynamic DAX measures with multi-select filters for stakeholder self-service analysis
- Identifies margin opportunities from raw inventory and sales data

## Files
| File | Description |
|------|-------------|
| `toniq.ipynb` | Data cleaning and preparation notebook |
| `toniq.xlsx` | Main retail dataset |
| `categories_export.xlsx` | Product categories data |
| `products_export.xlsx` | Products data |
| `inventory-export.xlsx` | Inventory export data |
| `Inventory Intelligence.docx` | Project documentation |

## Tools Used
- Python (Pandas) — data cleaning
- Power BI + DAX — dashboard and measures
- SQL — data querying
- Excel — raw data

## Key Result
Identified $30K+ in profit potential and tracked 36% average margins across product categories using real retail transaction data.
